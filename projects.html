Long Short-Term Memory Recurrent Neural Net (LSTM)

from-scratch C++

A class project for CS678-Advanced Nueral Networks and Machine Learning. It implements Gers, Schmidhuber, Cummins’ 2000 paper, “Learning to Forget: Continual Prediction with LSTM.” This is a single-hidden-layer network that allows both LSTM blocks and normal hidden nodes.

Touring into the Picture (TIP)

C++ and OpenCV

This is a project for CS750 Advance Computer Vision implementing the 1997 SIGGRAPH paper by Horry, Anjyo and Arai. The basic premise is creating a 3D environment from a 2D picture by leveraging perspective. In my program, the user specifies a backwall and a vanishing point. From this, five sides of a virtual “hallway” are created, allowing the user to move around to novel views of the scene. I did not implement foreground objects as the paper did.
